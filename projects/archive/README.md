# Projekt-Archiv

Abgeschlossene oder pausierte Projekte. Pro Jahr ein Unterordner (`2026/`, `2027/`, ...).

## Wann archivieren

- Projekt-Ziel erreicht und nichts mehr dran zu tun
- Projekt eingestampft (Pivot, kein ROI, …)
- Projekt für > 3 Monate pausiert

## Wie archivieren

```bash
mkdir -p projects/archive/2026
mv projects/active/<projekt-slug> projects/archive/2026/
```

## Warum behalten

Archivierte Projekte sind eine **goldene Quelle für Lessons Learned**. Wenn ein neues Projekt ähnlich aussieht, lohnt sich oft ein Blick ins Archiv.
