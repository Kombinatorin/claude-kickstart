# Inbox

Lose Tasks und Ideen — alles, was nicht sofort einem Projekt zugeordnet werden kann.

## Anleitung

- **Hinzufügen:** Schreibe einen Bullet `- [ ] Task-Beschreibung — Quelle, Datum`
- **Erledigt:** Box abhaken oder Zeile löschen
- **Promotion:** Wenn ein Task zu groß wird, in ein Projekt unter `projects/active/` ziehen
- **Aufräumen:** Beim `/weekly-review` aufräumen — alte/irrelevante Tasks raus

`/session-start` liest diese Datei am Anfang einer Session.
`/session-end` kann offene Punkte hier ergänzen.
