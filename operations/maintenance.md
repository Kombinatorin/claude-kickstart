# Maintenance

Recurring Ops — Dinge, die regelmäßig passieren müssen, aber kein eigenes Projekt sind.

## Format

```
## <Routine-Name>
- Frequenz: täglich / wöchentlich / monatlich / quartalsweise
- Verantwortlich: <Name oder du selbst>
- Letzte Ausführung: YYYY-MM-DD
- Notizen: <wenn was schief geht, hier eintragen>
```
