## Format pro Eintrag

```
## Woche YYYY-Wxx (Mo TT.MM – So TT.MM)

### Was erreicht
- Punkt 1
- Punkt 2

### Was nicht erreicht (und warum)
- Punkt 1 — Grund
- Punkt 2 — Grund

### Lessons Learned
- 1–3 echte Erkenntnisse, die das nächste Mal besser machen

### Fokus nächste Woche
- 1–3 konkrete Outcomes

### Energie / Stimmung (1–5)
Numerisch + 1 Satz.
```
