# Session End

Beende die Arbeitssession sauber.

## Lies zuerst

1. `operations/inbox.md`
2. `context/current-priorities.md`
3. Aktive Projekte (`projects/active/<x>/status.md`), falls in dieser Session daran gearbeitet wurde

## Gib diesen Abschluss aus

```
## 🏁 Session Ende — [DATUM] [UHRZEIT]

### ✅ Diese Session erledigt
(Was wurde konkret abgeschlossen? Maximal 5 Punkte, 1 Satz je Item)

### 🔄 Angefangen, aber nicht fertig
(Was ist halb fertig? Was braucht nächste Session Aufmerksamkeit?)

### ➡️ Nächster konkreter Schritt
(Genau 1 Satz: Was ist das Erste, was du beim nächsten Öffnen anpackst?)
```

## Danach (Schreiben)

- `operations/inbox.md`: erledigte Punkte abhaken, neu aufgetauchte Tasks ergänzen (jeweils mit Quelle + Datum)
- Aktives Projekt (falls bearbeitet): `projects/active/<x>/status.md` aktualisieren
- Wenn in der Session eine Entscheidung getroffen wurde: Eintrag oben in `decisions/log.md` (Format: `templates/decision-entry.md`)

**Wichtig:** Kein Fließtext. Konkret, ausführbar. Der nächste Schritt muss so klar sein, dass die nächste Session ohne Kontext sofort weitergehen kann.
