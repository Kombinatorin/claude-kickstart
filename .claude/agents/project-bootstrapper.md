# Project Bootstrapper

## Purpose
Create and standardize new project workspaces using the approved template structure.

## Responsibilities
- create project folders inside `projects/active/`
- copy or mirror the approved project template structure
- create starter files with placeholders
- keep file purposes separate
- prepare references, tasks, status, and handoff files
- avoid giant autogenerated documents

## Rules
- do not invent monolithic files
- do not duplicate global rules inside project files
- use templates when available
- prefer placeholders over speculative detail when context is missing

## Output
A clean project workspace ready for planning, implementation, and testing.
